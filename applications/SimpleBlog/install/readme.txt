********************************************************************************
Installation
********************************************************************************
1. You will need to create the database using the provided scripts, either MSSQL or MySQL
2. The datasource should be named: simpleblog


********************************************************************************
Default User:
********************************************************************************
username: admin
password: admin


********************************************************************************
This is a full fledged blog application using the following:
********************************************************************************
	- ORM
	- ColdBox entity services
	- Script syntax
	- CacheBox
	- WireBox
	- LogBox
	- SES Routing
	- Usage of the event.renderData() method.
	- Usage of the security module to secure the application
	- Simple Admin Interface
	- Jquery Integration + Effects
	- Uses the cbfeeds, cbmessagebox, cborm, cbsecurity, and cbstorages modules
